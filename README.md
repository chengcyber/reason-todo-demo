# Reason Todo

> Simple Todo Demo written in ReasonML

## Usage

```
git clone git@github.com:kimochg/reason-todo-demo.git
cd reason-todo-demo
npm run install
npm run start
npm run build
npm run server
```

visit `http://127.0.0.1:8080`
